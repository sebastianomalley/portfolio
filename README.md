# Jason O'Malley — Portfolio Site

## File Structure
```
jason-omalley/
├── index.html          ← Main page (hero + code projects + film projects)
├── about.html          ← About page
├── css/
│   └── style.css
├── js/
│   └── main.js
├── images/
│   └── headshot.jpg    ← ADD YOUR PHOTO HERE (rename to headshot.jpg)
└── resume/
    └── resume.pdf      ← ADD YOUR RESUME HERE (rename to resume.pdf)
```

## To Customize
1. Drop your headshot into `images/` named `headshot.jpg`
2. Drop your resume PDF into `resume/` named `resume.pdf`
3. Replace `#` placeholder links with real GitHub/demo URLs in `index.html`
4. Update email in footer of both HTML files
5. Update film links once hosted (Vimeo, YouTube, etc.)

## To Deploy on DigitalOcean
- Upload the entire folder to your DigitalOcean droplet or App Platform
- Point your domain (jason-omalley.com) to it
- No build step needed — pure HTML/CSS/JS
